A web application demonstrating API integration and async/await in JavaScript.

 Features
1. Fetching data from JSONPlaceholder API using Fetch with async/await
2. Parsing JSON response into JavaScript objects
3. Dynamic DOM manipulation - creating user cards programmatically
4. Error handling with try/catch blocks
5. Show/Hide toggle functionality with event listeners
6. Responsive grid layout with CSS

How to Run
Open `index.html` in your browser

API Used
https://jsonplaceholder.typicode.com/users
